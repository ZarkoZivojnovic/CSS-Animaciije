CSS Animacije

2D Transformacije

Transformacije same po sebi nisu animacije, ali u kombinaciji sa Transition mogu da izgledaju zanimljivo. Postoje sledeće metode:

  • translate()
  • rotate()
  • scale()
  • skewX()
  • skewY()
  • matrix()

Translate()

pomera element u odnosu na trenutnu poziciju.

transform:
translate
(25px,25px)

Rotate()

rotira element u zavisnosti od zadatog ugla.

transform:
rotate
(45deg)

Scale()

uvećava ili smanjuje veličinu elementa u zavisnosti od parametara


transform:
scale
(1.5,1.5)

SkewX()

iskosiće element po X osi.

transform:
skewX
(20deg)

SkewY()

iskosiće element po Y osi.


transform:
skewY
(30deg)

Matrix()

metoda je kombinacija svih 2D metoda u jednoj i parametri izgledaju ovako: matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())


transform:
matrix
(1, 0.3, 0.5, 1.5, 1, 2)

Spisak svih funkcija izgleda ovako: matrix(n,n,n,n,n,n), translate(x,y), translateX(n), translateY(n), scale(x,y), scaleX(n), scaleY(n), rotate(angle), skew(x-angle,y-angle), skewX(angle), skewY(angle).

3D transform osobine:

  • transform - primenjuje 2D ili 3D transformaciju na element
  • transform-origin - omogućava da promenite poziciju transformisanog elementa
  • transform-style - određuje kako da element bude prikazan u 3D prostoru
  • perspective - određuje perspektivu
  • perspective-origin - određuje donji položaj 3D elemeta
  • backface-visibility - definiše da li će element biti vidljiv

3D transform metode:

CSS Transitions

Da bi smo podesili transition efekat moramo definisati dve stvari: CSS osobinu na koju želimo da se promena primeni i vreme trajanja promene.

Primer:
div{width: 100px; height: 100px; background: red; transition: width 2s;}
div:hover{width: 300px;}

Može se primeniti i više promena u jednoj tranziciji.

transition: width 2s, height 4s;

Brzina kretanja tranzicije

Osobina transition-timing-function odredjuje oscilacije u brzini tranzicije

  • ease - definiše spori start, zatim ubrzanje i spori kraj tranzicije (default podešavanje)
  • linear - ista brzina od početka do kraja promene
  • ease-in - spori početak tranzicije
  • ease-out - spori kraj tranzicije
  • ease-in-out - spori početak i kraj tranzicije
  • cubic-bezier(n,n,n,n) - služi da sami definišemo funkciju
#div1 {transition-timing-function: linear;}
#div2 {transition-timing-function: ease;}
#div3 {transition-timing-function: ease-in;}
#div4 {transition-timing-function: ease-out;}
#div5 {transition-timing-function: ease-in-out;}

Odlaganje početka promene

Osobina transition-delay opisuje vreme (u sekundama) koje će proći pre početka promene

transition-delay: 2s;

Način pisanja CSS osobina

Prvi način je pisati svaku osobinu odvojeno.

transition-property: width;
transition-duration: 2s;
transition-timing-function: linear;
transition-delay: 1s;

Drugi način je da sve osobine pišemo u jednoj skraćenoj osobini.

transition: width 2s linear 1s;

Transition primeri:

CSS
div { transition: width 1s; }
div:hover {width: 200px}

div:hover {transform: rotate(360deg); background-color: purple;}

CSS

div:hover { color: yellow; font-size: 40px; transform: rotate(360deg);}

div:hover {opacity: 0.5; background-color: papayawhip; color: purple; transform: matrix(1, 0.3, 0.5, 1.5, 1, 2)}

Animations

Animacija omogućava elementu da se postepeno menja iz jednog stila u drugi. Možemo promeniti koliko hoćemo CSS osobina koliko god puta hoćemo. Da bismo koristili animacije prvo moramo definisati neke ključne okvire (keyframes). Keyframes sadrži stilove koje će element imati u određenom vremenu.

@keyframes pravila

Kada odredimo stilove unutar keyframes-a animacija će se postepeno menjati iz jednog stila u drugi. Da bi animacija funkcionisala moramo povezati animaciju sa elementom. Pogedajte primere:

@keyframes example {
from {background-color: red;}
to {background-color: yellow;}
}

div {
width: 100px;
height: 100px;
background-color: red;
animation-name: example;
animation-duration: 4s;
}
@keyframes example {
0% {background-color: red;}
25% {background-color: yellow;}
50% {background-color: blue;}
100% {background-color: green;}
}

div {
width: 100px;
height: 100px;
background-color: red;
animation-name: example;
animation-duration: 4s;
}

Osobine:
animation-duration- vreme trajanja animacije
animation-delay- odlaganje početka animacije
animation-iteration-count- određuje koliko će se puta animacija ponoviti
animation-direction- određuje da li će animacija ići unazad, unapred ili kombinovano. Može imati sledeće vrednosti: normal- unapred (default), reverse- unazad, alternate- prvi ciklus unapred, zatim unazad i alternate-reverse- prvi ciklus unazad, zatim unapred.
animation-timing-function - oscilacije u brzini animacije (vrednosti iste kao i kod tranzicija)

Osobine možemo pisati na dva načina:

div {
animation-name: example;
animation-duration: 5s;
animation-timing-function: linear;
animation-delay: 2s;
animation-iteration-count: infinite;
animation-direction: alternate;
}
animation: example 5s linear 2s infinite alternate;
